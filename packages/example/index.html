Getting Started

Start using Liquid Glass in minutes

1. Include the Stylesheet

Add the Liquid Glass stylesheet to your HTML:

HTML 
<link rel="stylesheet" href="liquid-glass.scss">

2. Add the SVG Filter (Optional)

For the lens distortion effect, include this SVG filter in your HTML:

HTML 
<svg xmlns="http://www.w3.org/2000/svg" style="display: none;">
  <filter id="lensFilter">
    <feComponentTransfer in="SourceAlpha" result="alpha">
      <feFuncA type="identity" />
    </feComponentTransfer>
    <feGaussianBlur in="alpha" stdDeviation="50" result="blur" />
    <feDisplacementMap in="SourceGraphic" in2="blur" scale="50"
                       xChannelSelector="A" yChannelSelector="A" />
  </filter>
</svg>

3. Create Your First Glass Element

Use the basic glass container structure:

HTML 
<div class="glass-container">
  <div class="glass-content">
    Hello, Glass!
  </div>
</div>

4. Customize with CSS Variables

Override CSS custom properties to customize the effect:

CSS 
.my-custom-glass {
  --lg-backdrop-blur: 8px;
  --lg-bg-color: rgba(0, 0, 0, 0.3);
  --lg-border-radius: 1rem;
}

Dark Glass

A darker glass effect perfect for dark mode interfaces.

Dark Mode
CSS 
.dark-glass {
  --lg-bg-color: rgba(0, 0, 0, 0.4);
  --lg-highlight: rgba(255, 255, 255, 0.3);
  --lg-backdrop-blur: 8px;
}

Strong Blur

Enhanced blur for a more pronounced frosted effect.

Enhanced Blur
CSS 
.strong-blur {
  --lg-backdrop-blur: 12px;
  --lg-filter-saturate: 150%;
}

Elevated Shadow

Deeper shadows create a floating effect.

Floating Effect
CSS 
.elevated-glass {
  --lg-shadow-offset-y: 12px;
  --lg-shadow-blur-primary: 12px;
  --lg-shadow-blur-secondary: 30px;
}

Minimal Glass

Subtle effect with reduced blur and transparency.

Subtle Effect
CSS 
.minimal-glass {
  --lg-backdrop-blur: 2px;
  --lg-bg-color: rgba(255, 255, 255, 0.15);
  --lg-shadow-blur-primary: 4px;
}
