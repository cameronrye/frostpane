# Frostpane Example Site

This is the demo/example site for the Frostpane library. It showcases various use cases and customization options.

## Running the Example

From the root of the monorepo:

```bash
npm install
npm run dev
```

Or from this package directory:

```bash
npm run dev
```

## Building the Example

```bash
npm run build
```

## Preview the Built Site

```bash
npm run preview
```

## What This Demonstrates

This example site shows:

- Basic glass container usage
- Different size variants (small, medium, large)
- Rounded and standard border radius options
- Music player component
- Navigation bar component
- Icon dock with lens filter effect
- Search button component
- Various customization examples using CSS custom properties
- Interactive elements with hover and active states
- Form inputs with glass styling
- Notification toasts
- Card grids

## How It Uses the Library

The example imports the Frostpane library as a dependency:

```javascript
import '@frostpane/core/scss';
```

This demonstrates how consumers of the library would use it in their own projects.
