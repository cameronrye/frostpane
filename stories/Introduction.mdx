# Frostpane Component Library

Welcome to the Frostpane Storybook! This interactive documentation showcases all the glass effect components and their variants.

## What is Frostpane?

Frostpane is a customizable, modern CSS/SCSS library for creating beautiful frosted glass effects with backdrop blur, highlights, and smooth animations. Perfect for creating frosted glass UI components with macOS-style aesthetics.

## Getting Started

### Installation

```bash
npm install @frostpane/core
```

### Basic Usage

```javascript
import '@frostpane/core/css';
```

### HTML Structure

```html
<div class="glass-container">
  <div class="glass-content">Your content here</div>
</div>
```

## Browse Components

Use the sidebar to explore different components and their variants:
